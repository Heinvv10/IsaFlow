# Module: Accounting

## Purpose
Core accounting module — AR, AP, banking, GL, reporting, assets, VAT, budgets, and company settings for South African businesses.

## Critical Rules
- ALWAYS scope queries with `company_id` (multi-tenant safety)
- ALWAYS use `sql` tagged templates from `@/lib/neon` (never string concat)
- ALWAYS wrap API handlers with `withCompany(withErrorHandler(handler as any))`
- NEVER use `console.log` — use `log` from `@/lib/logger`
- Customer invoices have BOTH `client_id` and `customer_id` — use `COALESCE(ci.client_id, ci.customer_id)` for joins
- Date columns from Neon return `Date` objects, not strings — always handle both types
